Python Basics

Python is a high-level, interpreted programming language known for its clear syntax and readability.

# Variables and simple operations
x = 5
y = 3
result = x + y
print(f"{x} + {y} = {result}")
  • Variables: No declaration needed. Just assign: x = 5
  • Data Types: Integers, floats, strings, booleans
  • Operators: Arithmetic (+, -, *, /), comparison (==, !=, >, <), logical (and, or, not)
  • Input/Output: print() and input() functions

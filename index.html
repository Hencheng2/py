Object-Oriented Programming

Python supports OOP with classes and objects.

# Class definition
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age
        
    def bark(self):
        return f"{self.name} says woof!"

my_dog = Dog("Rex", 3)
print(my_dog.bark())
  • Classes: Blueprints: class ClassName:
  • Objects: Instances: obj = ClassName()
  • Inheritance: Creating new classes from existing ones
  • Polymorphism: Common interface for different types

File Handling

Working with files for reading and writing data.

# Reading from a file
with open('file.txt', 'r') as file:
    content = file.read()
    print(content)

# Writing to a file
with open('output.txt', 'w') as file:
    file.write("Hello, World!")
  • Opening files: open(filename, mode)
  • Modes: 'r' for read, 'w' for write, 'a' for append
  • Context manager: Use with for auto-closing
  • Reading methods: read(), readline(), readlines()

Error Handling

Error handling with try-except blocks.

# Basic error handling
try:
    num = int(input("Enter a number: "))
    result = 10 / num
    print(f"10 / {num} = {result}")
except ValueError:
    print("That's not a valid number!")
except ZeroDivisionError:
    print("You can't divide by zero!")
  • Try-except: Catch and handle exceptions
  • Specific exceptions: Catch specific error types
  • Finally: Code that runs regardless of exceptions
  • Custom exceptions: Create your own exception types

Advanced Topics

Advanced Python concepts for experienced developers.

# Decorators
def my_decorator(func):
    def wrapper():
        print("Before function")
        func()
        print("After function")
    return wrapper

@my_decorator
def say_hello():
    print("Hello!")

say_hello()
  • Decorators: Modify functions without changing code
  • Generators: Create iterators with yield statements
  • Context Managers: Manage resources with with statements
  • Metaprogramming: Code that manipulates code
